This Site

This is a very basic and simple static site built using only HTML and CSS. It was writen in Brackets.io and does make use of Bootstrap 4. Images are provided by Lorem Picsom. This page is running on an AWS EC2 instance that is serving you this static page via NGINX webserver. SSL generated using certbot's Let's Encrypt and Google Cloud's DNS and DNS API. This projects code is hosted on my GitHub Repo.

Over time this should expand and provide more and more information and services to myself and others.

As of right now, this is very much so a work in progress.
